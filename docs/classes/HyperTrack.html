Static onTrackingStateChanged

  • onTrackingStateChanged(listener: ((trackingState: TrackingState) => void)): EmitterSubscription

  • Listen for changes of the Tracking State.

      if (trackingState === 'connected') {
        // ... ready to go
      }
    })
